Reflection

During this module I learned the fundamentals of web development using HTML and CSS. At the beginning I found it challenging to understand how different tags worked together to structure a webpage. Over time I became more confident in using semantic HTML elements such as header, nav, main and footer.

The design of my website focused on simplicity and readability. I selected a clean layout with clear navigation so users can easily move between pages. CSS grid was used on the projects page to display portfolio items in an organized layout.

CSS keyframe animations were used to create a subtle animated background gradient to improve visual engagement while maintaining readability.

One challenge I encountered was making the website responsive for mobile devices. I solved this by using CSS media queries and a hamburger menu. This allowed the navigation menu to adapt to smaller screens.
