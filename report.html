AS1 Reflection

During this module I learned the fundamentals of web development using HTML and CSS. At the beginning I found it challenging to understand how different tags worked together to structure a webpage. Over time I became more confident in using various HTML elements such as header, nav, main and footer.

The design of my website focused on simplicity and readability. I selected a clean layout with clear navigation so users can easily move between pages. CSS grid was used on the projects page to display portfolio item placeholders in an organized layout.

CSS keyframe animations were used to create a constant animated background gradient to improve visual engagement while maintaining readability.

One challenge I encountered was making the website responsive for mobile devices. I solved this by using CSS media queries anda bit of javascript to create a hamburger menu. This allowed the navigation menu to adapt to smaller screens.

Throughout the development process I validated my HTML and CSS using the W3C validation tools to ensure my code was error-free and followed best practices.
